# Портфолио
В данном репозитории представлены примеры файлов, которые я создавал в процессе своей трудовой деятельности для облегчения жизни себе и коллегам. Ниже представлены описания каждого файла и конкретных результатов от его использования.

## **Расписание 1 ППА.xlsx**
>Excel: ВПР, умная таблица, условное форматирование, проверка данных

*Автоматизировал форму сбора расписания пересдач с преподавателей, благодаря чему была **исключена работа по ручному устранению наложений и некорректного порядка проведения пересдач**:*
1. Оставил доступными для редактирования только те столбцы, которые должны менять преподаватели (дата, время и номер аудитории).
2. С помощью функции "Проверка данных" прописал варианты дат и времени, из которых могут выбирать преподаватели.
3. Создал столбец-"ключ" из названия группы, даты и времени проведения пересдачи. Применил к этому столбцу условное форматирование, показывающее дубликаты, что позволило выявлять наложения в процессе заполнения.
4. Создал столбец проверки порядка курсовых и зачетов/экзаменов. Если порядок нарушен, в этом столбце в соответвующих строчках появится надпись "Прием курсового стоит позднее ПА" и сработает условное форматирование.
5. Преобразовал формат документа в умную таблицу для упрощения переноса проверок между файлами разных факультетов. Это также исключило проблемы, связанные с разным размещением или количеством столбцов, за счет использования в формулах названий столбцов, а не адресов ячеек.

## **Единый табель посещения с начала семестра.xlsm**
>Excel: Power Query, ДВССЫЛ, условное форматирование

*Автоматизировал табель оценки посещаемости, благодаря чему была **исключена работа старост по ручной оценке уровня посещаемости студентов их групп, повышена точность расчетов, более точно определялись группы для повышения дисциплины**:*
1. Для каждого старосты была создана онлайн Excel таблица. В таблице доступны только ячейки для заполнения часов пропусков и ФИО студентов.
2. Посредством Power Query данные о часах пропусков импортируются из файлов старост в единый документ на соответствующий лист, где с ними проводятся вычисления.
3. В общую таблицу, расположенную на основном листе, выносятся результаты этих вычислений. Обращения к листам осуществляются посредством функции ДВССЫЛ и названия группы.
4. На основе этих данных формируется гистограмма с накоплением, а также определяется общий процент посещаемости студентов факультета

## **Матрица ТТПб.xlsx**
>Excel: ГПР, ВПР, РАЗД, режим разработчика
*Автоматизировал поиск компетенций и индикаторов дисциплины по учебному плану, благодаря чему были **повышены точность и качество проведения проверки заполнения документов, основанных на учебном плане (рабочих программ дисциплин, оценочных материалов и т.п.)**:*
1. Матрицу дисциплин из учебного плана дополнил компетенциями и индикаторами с их расшифровками.
2. Посредством ГПР в матрице осуществляется поиск дисциплины с соответствующим порядковым номером. Для удобства через режим разработчика добавил стрелки для переключения номера дисциплины.
3. Индикаторы выбранной дисциплины с помощью функции РАЗД распределяются на несколько строк. В каждой строке из индикатора выделяется компетенция (первая часть до точки). Для компетенций и индикаторов с помощью функции ВПР определяются их расшифровки.
4. Для каждого индикатора из матрицы определется количество вопросов, которые должны быть представлены в сопутствующих документах.
